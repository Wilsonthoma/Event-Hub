# EventHub - Frontend Only

A modern, responsive event management system frontend with glassmorphic UI design.

## Tech Stack

- HTML5
- CSS3 (Glassmorphic effects)
- Django Templates

## Quick Start

```bash
git clone https://github.com/Wilsonthoma/Event-Hub.git
cd Event-Hub
python -m http.server --directory events/templates/
```
